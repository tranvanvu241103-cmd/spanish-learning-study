How this canvas works

This canvas is now organized as a real lesson system. Each lesson has its own vocabulary, pronunciation, grammar explanation in English, listening practice, and exercises using the same words.
  1. Start with 📚 Lesson 1.
  2. Listen to the vocabulary and repeat aloud.
  3. Study the grammar explanation in English.
  4. Do the listening tasks using only words from the lesson.
  5. Finish the exercises and check your score.
  6. Use ♻️ Reuse to add new words for future lessons.
Audio uses your browser voice system. For the best result, open this file in Chrome or Edge. Choose a Spanish voice if your browser asks.
